# Mapping Air Quality — project website

Static website + WebGIS for the GIS laboratory (Romania, 2021–2023).

## Pages
| Page | Purpose |
|------|---------|
| `Index` | Home — overview, study area, scorecard, objectives |
| `Methodology` | Datasets + the 8-step processing pipeline |
| `Results` | Results — figure / chart / table placeholders + analysis |
| `Webgis` | Interactive OpenLayers map |

## Project Structure
```
images/   | static maps & charts as .jpg   (Fig. 1–8)
css/      | Styling sheet
js/       | Button functionality and WebGIS interaction layer code and WMS server conections.
```
